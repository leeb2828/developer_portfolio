Public API Requests

Fetch random employees using the Random User API and display them in a gallery. You can search employees via the search box in the top-right corner of the page. Employees will hide or reappear, depending on your search results. If you click directly on any of the 12 employees in the gallery, a single modal window with more information about the employee you clicked on, will appear. You can use the "NEXT" and "PREV" buttons to toggle back and forth between employees when the modal window is open. Every time you refresh the page, a different set of 12 employees will appear. This was built using HTML, CSS, Flexbox, and JavaScript. I wrote all of the JavaScript for this project.
